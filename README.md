# IMAGE-ENHANCEMENT

## Image Smoothing and Sharpening Using OpenCV
This project demonstrates different image smoothing and sharpening techniques using Python and OpenCV. The program applies multiple filtering methods to enhance image quality, reduce noise, and improve edge details for image processing applications.

## Features
Read and display an input image
Apply Averaging Filter
Apply Weighted Averaging Filter
Apply Gaussian Filter
Apply Median Filter
Apply Laplacian Sharpening using Kernel
Apply Laplacian Operator for edge detection
Display all processed images for comparison
## Technologies Used
Python 3.7
OpenCV (cv2)
NumPy
Matplotlib
Jupyter Notebook / VS Code
Filters Implemented
Smoothing Filters
Averaging Filter
Produces a blurred image by averaging neighboring pixel values.

## Weighted Averaging Filter
Provides smoother results with reduced distortion using a weighted kernel.

## Gaussian Filter
Reduces noise while preserving image edges effectively.

## Median Filter
Removes salt-and-pepper noise efficiently.

## Sharpening Filters
Laplacian Kernel Sharpening
Enhances edges and fine details in the image.

## Laplacian Operator
Detects edges clearly in grayscale images.

## Algorithm
Import required libraries
Read the input image
Convert image from BGR to RGB format
Apply Averaging Filter
Apply Weighted Averaging Filter
Apply Gaussian Filter
Apply Median Filter
Apply Laplacian Sharpening using Kernel
Apply Laplacian Operator
Display all outputs using Matplotlib
